# Prithic Portfolio (Version 1)

This branch contains the **standalone HTML version** of my portfolio. It's a "vibe-coded" interactive dashboard featuring a futuristic, sci-fi aesthetic with terminal-style boot screens, neural core scans, and systemic health indicators.

## Features

- **Futuristic UI**: High-contrast, neon-themed design using `Orbitron` and `Rajdhani` typography.
- **Interactive Layers**: 
  - Real-time cursor glow effect.
  - Scanline overlays and animated number glitching.
  - "System Scan" dashboard representing core skills and personal milestones.
- **Standalone Design**: The entire portfolio is contained within a single `prithic_portfolio.html` file, including CSS and JavaScript, allowing for easy distribution and instant loading.

## Usage

Simply open `prithic_portfolio.html` in any modern web browser to see the interactive portfolio.
